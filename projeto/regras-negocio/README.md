# regras-negocio/

Tudo sobre **como o negócio funciona**.

## O que jogar aqui

- Regras de negócio (RNs) escritas
- Glossário de termos do domínio
- Fluxos de processo (BPMN, fluxogramas, textos descritivos)
- Casos de uso e user stories
- Personas
- Jornadas do usuário
- SLAs, prazos, políticas
- Restrições regulatórias específicas

## Formatos aceitos

`.md`, `.txt`, `.pdf`, `.docx`, `.xlsx`, `.csv`, `.png` (de fluxogramas), `.yaml`, `.json`

## Exemplos

```
rn-001-cobranca-tier-1.md
rn-002-cobranca-tier-2.md
glossario.md
fluxo-checkout.png
persona-cliente-pj.pdf
sla-suporte.txt
jornada-onboarding.md
```

## O que NÃO jogar aqui

- Mockups visuais (vai em `wireframes/`)
- Tokens de design (vai em `identidade-visual/`)
- Documentação de APIs externas (vai em `docs-externos/`)
