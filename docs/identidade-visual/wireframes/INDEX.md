> Pasta para wireframes navegáveis (.html, .jsx, .tsx, .svg, .pdf). Claude lê estes arquivos como referência de **layout e hierarquia** ao implementar sprints.

## Convenções

- Um arquivo por tela ou por fluxo (não jogar tudo em um HTML gigante)
- Nome descritivo: `<feature>-<variante>.<ext>` — ex: `create-listing-mobile.html`, `checkout-desktop.pdf`
- Se gerado por IA (v0, Lovable, Bolt), anotar no INDEX qual ferramenta e prompt usado (útil para regenerar)
